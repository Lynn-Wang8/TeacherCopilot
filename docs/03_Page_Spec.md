# 二、整体设计原则
## 2.1 产品定位
Teacher Copilot 是一个 AI 工作台（Workspace）。
不是聊天机器人。
不是AI搜索。
也不是普通管理后台。
整个产品围绕：
"整理错题"
这一件事情展开。
________________________________________
## 2.2 整体设计风格
关键词：
•	极简（Minimal）
•	专业（Professional）
•	教师工具（Teacher Tool）
•	AI Native
•	高信息密度
•	易编辑
参考风格：
•	ChatGPT
•	Linear
•	Notion
•	飞书文档
避免：
•	花哨动画
•	渐变背景
•	大面积插画
•	游戏化设计
整体色调：
白色 + 蓝色强调色 + 灰色辅助色。
________________________________________
## 2.3 布局原则
整个产品采用：
Desktop First
推荐宽度：
1440px
内容区域：
1200px
统一采用：
16px 间距系统。
所有页面保持一致导航逻辑。

# 十一、设计约束（供AI生成网页）
页面风格必须满足：
✅ 极简
✅ 中文界面
✅ 白色背景
✅ 蓝色强调色
✅ 卡片式布局
✅ 三栏工作台
✅ 高信息密度
✅ 专业教师工具风格
禁止：
❌ 游戏化
❌ 渐变背景
❌ 花哨动画
❌ AI聊天界面
❌ 大面积留白
整个页面应更接近：
ChatGPT Workspace + Notion + 飞书文档
而不是营销官网。
________________________________________
# 十二、验收标准（Acceptance Criteria）
完成以下功能视为MVP页面完成：
•	可上传试卷图片
•	可查看AI分析进度
•	AI完成后生成题型分类
•	支持拖拽调整题型
•	支持编辑易错点与教师备注
•	支持新增题型
•	支持修改导出文件名
•	成功导出Word讲义
页面整体应满足：
让一位从未使用过本产品的初中数学教师，在3分钟内完成第一次错题整理。
